#  Hotel Full System

Информационная система для управления гостиничным комплексом

##  Функциональность

- Управление номерным фондом
- Бронирование номеров  
- Регистрация гостей
- Заселение/выселение
- Поиск свободных номеров
- Отчетность и статистика

##  Технологии

- Python 3.8+
- Flask 2.3.3
- SQLite
- HTML5/CSS3
- Bootstrap 5
